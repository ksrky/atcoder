# ksrk's AtCoder archives

### Environment
* OS: Ubuntu 20.04
* Editor: CLion

## Create project of a contest
```commandline
chmod 755 setup.sh
./setup.sh contest_name
```
contest_name will be abc226, agc055 ...

## Test your code
```commandline
chmod 755 cptest.sh
./cptest.sh problem_name
```
problem_name will be a, b, c, d, e, f
